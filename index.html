Mission Brief · §01

The daily signal room for four channels of loose-cannon commentary.

NOD News is a daily desk that pulls the news, filters it through the four modes of the Mix & Match software — HAGGLE, FRIENDSHIP, BOSS MODE, ROGUE — and outputs ready-to-film angles for each. The dashboard is one screen. Every morning at 06:00 CT, twelve stories are queued (three per mode) with a source, a headline, and a loose-cannon angle. One click sends a story to the script generator.

This file is the handoff spec. It doubles as the visual reference for what the interface should look like. Everything below the fold is machine-readable configuration for the agent to consume, cache, and update daily.

PURPOSE    Daily content ideation across 4 modes
OUTPUT     12 story cards, 4 mode digests, 1 top pick
CADENCE    06:00 CT · Mon–Sat
HOST       Local iMac · localhost:3000
STACK      Static HTML + JSON feed
UPDATED    On cron + on manual refresh
RULE       Attack ideas, not identities

Build Spec · §03

What to build. In this order.

The frontend

  • stack Static HTML/CSS/JS or Next.js if the agent prefers
  • host Local iMac at localhost:3000
  • layout This file is the reference. Match the grid.
  • refresh Auto-poll /feed.json every 5 min
  • manual Header button forces refresh
  • archive "Yesterday's brief" link, up to 30 days

The backend

  • runtime Node or Python — agent's call
  • schedule Cron @ 05:45 CT weekdays + Sat
  • flow Fetch → filter → score → angle → write JSON
  • storage /data/YYYY-MM-DD.json + /data/latest.json
  • llm Use Claude API for angle generation
  • fallback If a mode returns <3 stories, backfill from prior day + flag

Story selection logic

  • volume Pull ~40 candidates per mode, cut to top 3
  • recency Prefer <24hr; hard cutoff 72hr
  • scoring See §04 — one formula per mode
  • de-dupe Cosine similarity ≥ 0.85 = drop the newer one
  • safety Enforce YouTube guardrails from brand bible
  • bias Prefer stories with public receipts + hard numbers

Angle generation

  • prompt "Write a one-sentence loose-cannon angle for this story, in the voice of [MODE]."
  • length ≤ 20 words per angle
  • tone Match the brand bible's mode-specific voice
  • rule Attack ideas, not identities. Enforce in prompt.
  • variants Generate 3, keep the sharpest, log the rest
  • bonus Also generate a title in that mode's title formula

Guardrails · §06

The rules that keep this monetized.

Every angle the agent generates has to pass one test: attack ideas, not identities. A CEO's strategy can be a scam. A CEO cannot be called a scammer without a conviction on file. Same rule for celebrities, founders, influencers, and public figures across all four modes. The agent's angle-generation prompt should include this line verbatim: "Critique the pattern, the product, the public statement, or the system. Never critique the person's identity, body, health, or existence."

✅ ALLOWED
Public actions, public statements, public numbers
Transformative commentary on public figures
Fair-use images for commentary
Profanity, strong emotion, "not financial advice"
❌ BLOCKED
Slurs, hate speech, protected-group attacks
Body-shaming, mental-health speculation
Election / health / medical misinformation
Criminal accusations without a conviction
Content involving minors, ever
